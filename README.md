# 医美纠纷消费者维权数据新闻（2024-2025）
基于中国裁判文书网140条检索数据（关键词：“医疗美容”“医美”“消费者”），筛选出108条消费者与医疗机构的裁判文书数据，制作交互式数据新闻。

## 数据来源
- 平台：中国裁判文书网
- 检索关键词：“医疗美容”“医美”“消费者”
- 时间范围：2024年1月-2025年11月
- 筛选标准：消费者作为原告、被告为医疗机构（民营/公立/工作室）、裁判文书完整

## 文件说明
- index.html：主页，含所有章节和图表容器
- style.css：自定义样式，优化移动端适配和交互体验
- js/chart-init.js：交互式图表脚本（基于Chart.js）
- data/cases.json：108条完整案件数据（脱敏处理）
- data/chart-data/：图表专用简化数据（用于渲染图表）

## 交互说明
- 图表支持hover显示详情（案件数、占比、金额等）
- 部分图表点击可查看典型案例详情
- 页面适配手机、平板、电脑等多终端

## 技术栈
- HTML5 + Tailwind CSS（响应式布局）
- Chart.js（交互式数据可视化）
- GitHub Pages（静态页面部署）
